# oldmachines.github.io

The top-level index of the [**oldmachines**](https://github.com/oldmachines)
collection — interactive, explorable teardowns of the machines we grew up
with. This repo serves the landing page at
**https://oldmachines.github.io/**; each machine lives in its own repository
and publishes beneath it:

| Machine | Year | Site | Repository |
| --- | --- | --- | --- |
| Apple II | 1977 | [/apple2/](https://oldmachines.github.io/apple2/) | [`apple2`](https://github.com/oldmachines/apple2) |
| Sega Genesis | 1988 | [/genesis/](https://oldmachines.github.io/genesis/) | [`genesis`](https://github.com/oldmachines/genesis) |
| Super Nintendo | 1990 | [/supernintendo/](https://oldmachines.github.io/supernintendo/) | [`supernintendo`](https://github.com/oldmachines/supernintendo) |
| GameCube | 2001 | [/gamecube/](https://oldmachines.github.io/gamecube/) | [`GameCube`](https://github.com/oldmachines/GameCube) |

It's a single dependency-free static page: no build step, no framework, no
tracking. Open `index.html` in a browser and it runs.

## Adding a machine

Add a card to `index.html` linking to the new repo's Pages path, and a row to
the table above. That's it.
